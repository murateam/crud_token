api-token-auth

If you know the username and the password you can get a token for this user

For example you can use the "cURL":

curl -d "{\"username\":\"username\", \"password\":\"password\"}" -H "Content-Type: application/json" -X POST http://[HOST NAME]/api-token-auth/

api

For these requests you don't need the token

For example you can use the "cURL":

curl -H "Content-Type: application/json" -X GET http://[HOST NAME]/api/v1/users/ # to get the list for exist users

curl -d "{\"username\":\"username\", \"password\":\"password\", \"is_active\":true}" -H "Content-Type: application/json" -X POST http://[HOST NAME]/api/v1/users/ # to crate a new user

api_v1_users_create

For these requests you need the token

For example you can use the "cURL":

curl -d "{\"username\":\"username\", \"password\":\"password\", \"is_active\":true}" -H "Content-Type: application/json" -H "Authorization: Token [USER TOKEN]" -X PUT http://[HOST NAME]/api/v1/users/{id}/ # to change the exist user

curl -H "Content-Type: application/json" -H "Authorization: Token [USER TOKEN]" -X DELETE http://[HOST NAME]/api/v1/users/{id}/ # to delete the user
